Query Parameters

  • offset: (integer - maximum: 199)

    Returns records that match your query starting with response number offset +1.

    Examples:

    10
  • limit: (integer - default: 20 - maximum: 200)

    Number of records per page.

    Examples:

    50
  • sortBy: (string - default: email)

    Sorting field ascending, by default sort by email

    Examples:

    john@gmail.com
  • email: (string)

    Examples:

    jo

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • firstName: required (string)
  • lastName: required (string)
  • email: required (string)
  • password: required (string)

    Password must contains at least one upper case letter, one lower case letter and one digit. The minimum password's size is 8 and maximum is 255.

Examples:

{
  "firstName": "John",
  "lastName": "Doe",
  "email": "john@example.com",
  "password": "Password1"
}
